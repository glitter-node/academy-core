# Academy Core Template

Self-contained user template for GnuBoard7.

## Features
- Academy-oriented information architecture
- Independent template bundle (no core dependency)
- Installable via template manager
- Supports 9 routes:
  /, /about, /programs, /instructors, /timetable, /notices, /faq, /consultation, /contact

## Status
Alpha (v0.1.0)

## Requirements
- GnuBoard7

## Notes
- No core modifications required
- Data binding and admin integration planned next
